# ft_printf

42 project which aims to recreate printf function.

## Supported

Convertions : sSpdDioOuUxXcC  
Flags : #0-+ and space  
Precision  
Field width  
Length modifier : hh h l ll j z  
Double %  

## Compiling

You may compile the library `libftprintf.a` by running `make`.

## Usage

Include this header in a C project :  

`#inlcude "ft_printf.h"`
